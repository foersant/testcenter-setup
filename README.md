# iqb-tba-docker-setup
dockerize(testcenter + testcenter admin + backends)

# run

```
git clone --recursive https://github.com/iqb-berlin/iqb-tba-docker-setup.git
cd iqb-tba-docker-setup
docker-compose up

```
